# Data-Analysis

## In this repository task 1, task 2 and task 3 are done in which following activities are done .
## Task 1 : Data Cleaning and Preprocessing 
  Loading data using pandas,
  Checking missing values,
  Handling incorrect data types,
  Cleaning numeric columns
  
## Task 2 : Exploratory Data Analysis
  Summary statistics(Mean, Median, Mode, Standard deviation),
  Visualization(Histogram, Boxplot, Scatter Plot),
  Correlation analysis

## Task 3 : Basic Data Visaulization
  Bar plots, line charts, scatter plots, 
  Title, size, labels customization and exporting images.
